About

Kerry D. Rosado

I live in San Jose, CA and I am a Full Stack Web Developer with a B.S. in Computer Science. I specialize in HTML, CSS, JavaScript, and jQuery. I also have experience with Bootstrap, Python, Angular.js, Node.js, Express.js, and Git. I am also a member of Women Who Code, Girl Develop It, and PyLadies.

In my spare time I enjoy dancing Salsa, Merengue, Ballroom, and Tango. I am part of a performance team at Dance Blvd in San Jose, CA and will be performing in their Masquerade Ball and the Winter Showcase. I also enjoy symphonies, art exhibits, musicals, operas, and comedy shows.

Portfolio

Shopping List

A web application that creates a shopping list. It allows the user to add, delete, and clear the list. Technologies used were HTML, CSS, JavaScript, and jQuery.

Hot or Cold Game

A number guessing game where the user has to guess a number between 1 and 100. As they submit their guesses the game lets them know how hot or cold the are to the correct response. Technologies used were HTML, CSS, JavaScript, and jQuery.

Fine Wine Quiz

A quiz app that asks users multiple choice questions on fine wine, while tracking the number of correct responses. Technologies used were HTML, CSS, JavaScript, and jQuery.

PicSearch

An API project that uses the Instagram API to search for pictures. The app allows users to view more pictures or clear the page. Technologies used were HTML, CSS, JavaScript, and jQuery.

Animal Rescue

A hackathon project created as part of a 4 person team, which won second place. This web application helps animals find homes. Technologies used were HTML, CSS, JavaScript, jQuery, Bootstrap, and Meteor.

Random Quote Machine

A web application that displays random quotes. Technologies used were HTML, CSS, Bootstrap, JavaScript, and jQuery.

Pomodoro Clock

A timer that allows the user to control the length of session and breaks. Technologies used were HTML, CSS, Bootstrap, JavaScript, and jQuery.

Wikipedia Viewer

A Wikipedia viewer that allows the user to enter and item in search box, then displays results. Technologies used were HTML, CSS, JavaScript, jQuery, and Bootstrap.

Twitch API

A web application that uses the Twitch API. The app allows users to click the status output and be sent to the channel selected. It also lets you view what the user is streaming. Technologies used were HTML, CSS, JavaScript, jQuery, Bootstrap, and Twitch API.

Voting App

A voting application that allows the user to create, delete, and share polls. The user can also create new options for the polls and view results in chart form. Technologies used were HTML, CSS, Bootstrap, JavaScript, and jQuery.

Nightlife Coordination

A nightlife coordination application that allows the user to view bars in the area. Plus, they can tag or untag themselves at a location. Technologies used were HTML, CSS, Bootstrap, JavaScript, and jQuery.

Stockmarket App

A stockmarket application that allows the user to view a graph displaying the recent trend lines for each added stock. The user can also add new stocks, remove stocks, and view changes in real time. Technologies used were HTML, CSS, JavaScript, jQuery, and Bootstrap.
